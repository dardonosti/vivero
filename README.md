# Vivero
Trabajo práctico: Taller de Programación IV.

#### Instalar depedencias en el backend:

```
$ cd backend
$ mvn package
```

#### Instalar dependecias en el frontend:

```ash
$ cd frontend
$ npm install
```

#### Eliminar depedencias en el backend:

```
$ cd backend
$ mvn clean
```

#### Eliminar depedencias en frontend:

```
cd frontend
rm -r node_modules
```
