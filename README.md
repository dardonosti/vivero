# Vivero
Trabajo práctico: Taller de Programación IV.

#### Instalar dependencias en backend:

```
$ cd backend
$ mvn package
```

#### Instalar dependencias en frontend:

```ash
$ cd frontend
$ npm install
```

#### Eliminar dependencias en backend:

```
$ cd backend
$ mvn clean
```

#### Eliminar dependencias en frontend:

```
$ cd frontend
$ rm -r node_modules
```
